【図情報(プレゼンテーション)概要】

プレゼンテーション基底インターフェースはIPresentationで、それを継承するINodePresentation、ILinkPresentation、ICellがあります。 ICellはCRUDのプレゼンテーションのインターフェースで、それをIHeaderCell、IValueCellが継承しています。
