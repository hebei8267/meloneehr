【トランザクション操作(モデル/図/図要素の作成・編集・削除時)】

次の操作を行う場合は、トランザクション操作が必要です。
・モデルを作成・編集・削除する場合
・図を作成・編集・削除する場合
・図要素を作成・編集・削除する場合

トランザクションの開始：TransacionManagerのbeginTransactionメソッド
トランザクションの終了：TransacionManagerのendTransactionメソッド
トランザクションのアボート：TransacionManagerのabortTransactionメソッド

※astah* APIのモデル/図/図要素の作成・編集・削除機能は、astah* UML, professionalでサポートしています。
