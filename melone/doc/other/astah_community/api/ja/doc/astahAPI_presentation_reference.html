astah* API 利用ガイド

【図要素の情報の取得】

astah* APIの図情報の取得は、astah* UML, professionalでサポートしています。

【図から図要素の情報を取得】

図(IDiagram)のメソッドgetPresentations()を利用して、 図に配置されたプレゼンテーション、および、CRUDの各セル(値のセルおよびヘッダーセル)を取得します。

ProjectAccessor prjAccessor = ProjectAccessorFactory.getProjectAccessor();
prjAccessor.open("C:a.asta", true, false, true);
IModel project = prjAccessor.getProject();
IDiagram[] diagrams = project.getDiagrams();
for (int i = 0; i < diagrams.length; i++) {
    IPresentation[] iPresentations = diagrams[i].getPresentations();
}

【モデルから図要素の情報を取得】

モデル(IElement)のメソッドgetPresentations()を利用してプレゼンテーションを取得します。
  • オブジェクト図、ステートマシン図、アクティビティ図、シーケンス図、コミュニケーション図のモデルとプレゼンテーションは1対1です。
  • ノートアンカーやフレーム等モデルのないプレゼンテーションがあります。
