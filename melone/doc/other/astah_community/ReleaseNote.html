【新規機能】

  • ドロー・サジェスト対応要素の追加
    • シーケンス図：同期メッセージ、非同期メッセージ
      • アイコンボタンは、ライフライン、実行仕様、相互作用の利用、フレーム上をマウスオーバーして表示します。
    • コミュニケーション図：メッセージ
      • アイコンボタンは、リンク上をマウスオーバーして表示します。但し、リンク線のスタイルが曲線又は直角曲線の場合は表示できません。
  • キーバインドファイルの対応【※Macは対象外です】
    • astah*で使用するショートカットキーをカスタマイズできるようになりました。
      • [方法]
        • 1. astah*を一旦終了します。
        • 2. astah*インストールフォルダ配下の"astah-key.properties_org"を、ユーザーホーム\.astah\community配下にコピーして、"astah-key.properties"という名前で保存します。
        • 3. astah-key.propertiesを開き、カスタマイズしたいショートカットキー(xxx.key)の#を削除してキーの値を変更します。
        • 4. astah-key.propertiesを保存後、astah*を起動します。
      • [注意]
        • キーが重複した際の動作は保証しません。
        • デフォルトで、同一のショートカットキー(xxx.key)を持つ異なる操作がございます。これらは常に同じキーに統一してください。
        • astah*のバージョンアップ、又は再インストールで、astah*を上書きインストールした場合は、新しい"astah-key.properties_org"と"astah-key.properties"の差分を自身で調整してください。
  • ショートカットキーの変更
    • クラスに属性を追加するショートカットキーが、Ctrl+FからCtrl+Rに変わりました。
  • ダイアグラムエディタ上の文字列編集時の自動スクロール
    • 図上で文字列を編集中、主要な要素に限り自動スクロールするよう改善しました。
  • 図上ダブルクリックで作成可能な図要素の追加
    • 合成構造図の構造化クラス内をダブルクリックして、パートを作成できるようになりました。
  • ソフトウェア更新確認の改善 (Windowsのみ)
    • ソフトウェア更新確認用のオプションをシステムプロパティに追加しました。
      • [システムプロパティ] - [更新確認] - [Internet Explorerの設定からプロキシ情報を自動で使用する(Windowsのみ)]: デフォルトON
  • APIの改善
    • アイコンの定義IconDescriptionの追加
    • 図ビューを管理するオブジェクトIDiagramViewManagerの追加
    • アイコンを管理するオブジェクトIIconManagerの追加
    • ビューを管理するオブジェクトIViewManagerの追加
    • ProjectAccessorにIViewManager getViewManager()メソッドを追加。ビューを操作するインターフェースを返します。
    • IAttributeにIConnector[] getConnectors()メソッドを追加。コネクタを取得します。
    • ProjectAccessorにエディション、バージョン、モデルバージョンを取得するメソッドを追加
      • String getAstahEdition()
        • astah*本体のエディションを取得します。
      • String getAstahVersion()
        • astah*本体のバージョンを取得します。
      • int getProjectModelVersion()
        • 現在開いているastah*プロジェクトのモデルバージョンを取得します。
      • int getAstahModelVersion()
        • 現在開いてるastah*のモデルバージョンを取得します。
      • String getAstahAPIVersion()
        • astah*APIのバージョンを取得します。
      • int getAstahAPIModelVersion()
        • astah*APIでモデルを保存する場合のモデルバージョンを取得します。
